# 💜 Eu Te Escuto

Projeto desenvolvido com foco em apoio emocional, informação e conscientização sobre relacionamentos abusivos e saúde mental.

Este projeto nasce da união entre minha experiência profissional na área da saúde (enfermagem e psicanálise) e meu processo de aprendizado em desenvolvimento web.

---

## 🧠 Objetivo

Criar uma plataforma digital que ofereça acolhimento, informação e suporte emocional para pessoas que estejam passando por momentos difíceis, especialmente em contextos de relações abusivas e dependência emocional.

---

## 🚀 Funcionalidades

Neste projeto, busquei ir além do básico aprendido em sala de aula, implementando:

- Página inicial com conteúdo informativo
- Seção “Sobre nós” com contexto profissional e propósito do projeto
- Página sobre Narcisismo (educacional)
- Biblioteca com materiais para download (e-books)
- Vídeos explicativos incorporados
- Formulário de contato
- Botão de contato direto via WhatsApp
- Navegação entre páginas

---

## 📈 Evolução em relação ao primeiro projeto

Este é o segundo projeto desenvolvido durante o primeiro semestre, e representa uma evolução clara em relação ao primeiro projeto (**loja virtual / techstore**).

### 🔹 No primeiro projeto:
- Estrutura mais simples
- Foco em HTML e CSS básico
- Poucas interações
- Layout mais limitado

### 🔹 Neste projeto:
- Maior organização de páginas
- Inclusão de múltiplas funcionalidades
- Integração com WhatsApp
- Uso de vídeos e conteúdos dinâmicos
- Criação de biblioteca de materiais
- Maior preocupação com o usuário e propósito do site

---

## ⚠️ Pontos de melhoria
